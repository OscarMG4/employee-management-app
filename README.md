### Frontend (React + TypeScript)

1. Clonar el repositorio del frontend
2. Navegar a la carpeta del proyecto
3. Instalar las dependencias con: npm install
4. Copiar el archivo .env.example a .env
5. Iniciar el servidor de desarrollo con: npm run dev

## Funcionalidades

### Módulo de Departamentos
- Crear, editar y eliminar departamentos
- Estructura jerárquica de departamentos (padre-hijo)
- Búsqueda por nombre o embajador
- Filtros por nivel, división superior y número de colaboradores
- Ordenamiento por múltiples columnas
- Paginación configurable
- Vista de lista y árbol

### Módulo de Empleados
- Gestión completa de empleados
- Asignación a departamentos
- Información de contacto y laboral
- Búsqueda y filtros avanzados
- Estadísticas en tiempo real

## Tecnologías Utilizadas

Backend: Laravel 12, MySQL, PHP 8.2
Frontend: React 19, TypeScript, Ant Design, Vite, TanStack Query
Estilos: Ant Design Components
